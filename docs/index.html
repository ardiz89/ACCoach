FAQ

Do I need an account or a connection?

No. HONE is 100% offline: everything stays on your PC, nothing is sent to third parties. Laps are saved under Documents\ACCoach\.

Windows warns me with “SmartScreen” — is it safe?

Yes. The executable isn't signed with a paid certificate yet, so Windows shows a notice on first run. Click “More info → Run anyway”. You can verify the file by comparing its SHA-256 with the one published in the release. Full steps in the FAQ.

Does it work with plain Assetto Corsa, or only ACC?

Both. HONE reads the shared memory common to AC and ACC. The setup engineer has profiles for low-downforce road cars too, not just GT3.

The overlay doesn't show over the game

Set the game to Borderless (windowed-fullscreen): a transparent overlay can't draw over exclusive fullscreen. Same constraint as SimHub and Crew Chief.

How much does it cost?

It's free and complete today. The model is one-time freemium: Free stays free forever, Pro will be a one-time purchase (no subscription).
